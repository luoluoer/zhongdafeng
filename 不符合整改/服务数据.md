## 七、知识库条目（KB-CD-NET-003）

| 项目 | 内容 |
|------|------|
| **知识编号** | KB-CD-NET-003 |
| **标题** | 充电桩扫码后无法启动充电的快速排查指南 |
| **关键字** | 扫码失败，充电启动失败，通信超时，充电桩离线 |
| **适用设备** | 120kW/360kW直流充电桩，480kW充电堆 |
| **现象** | 用户扫码后APP提示“启动充电失败”或“设备通信超时”，桩屏幕显示正常但无法充电。 |
| **排查步骤** | 1. **检查桩在线状态**：登录运营平台查看该桩是否在线，若离线检查4G信号或网线连接；<br>2. **查看桩端告警灯**：红色告警灯常亮表示设备故障，需现场处理；<br>3. **远程查看日志**：登录桩端管理系统，查看主控与充电模块通信日志，确认是否有超时或报错；<br>4. **检查模块状态**：若日志显示模块通信失败，现场检查模块电源指示灯、风扇状态，判断是否模块故障；<br>5. **重启尝试**：远程重启主控板，若恢复但问题反复，可能为固件兼容性问题，需升级固件；<br>6. **检查平台版本**：确认平台最近是否有升级，如有可参考KB-CD-FW-006进行兼容性排查。 |
| **关联事件** | 基于INC-YD-20251120-008、INC-YD-20251108-012等事件总结 |
| **创建/更新人** | 王海/2025年11月25日 |
